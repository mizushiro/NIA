로그인 방식 오픈 API 호출 예

다음은 아파트엔 회원 프로필을 조회하는 API를 호출하는 코드를 Java로 작성한 예입니다.
아파트엔 앱에 로그인해서 획득한 접근 토큰을 요청 헤더에 추가해 프로필 조회 API를
RESTful API 방식으로 호출합니다. 반환받은 결괏값은 JSON 형식으로 출력합니다.

쉽고 간편한 이용 신청 / 관리 과정

서비스 절차 (신청, 승인 절차)

이용관리

리전별로 이용 중인 서비스와 보유 계약 현황을 실시간으로 확인하여 관리할 수 있습니다.
이용 현황을 확인하는 방법은 다음과 같습니다.

1. 아파트엔 클라우드 플랫폼 포털에 접속해 주십시오.
2. 아파트엔 클라우드 플랫폼 포털 화면이 나타나면 우측 상단에 있는 로그인을 클릭해 주십시오.
3. 로그인 화면이 나타나면 아이디와 비밀번호 입력 후 [로그인] 버튼을 클릭해 주십시오.
4. 중앙 상단에 있는 마이페이지 메뉴를 클릭한 다음 이용관리 메뉴를 클릭해 주십시오.
5. 이용관리 화면이 나타나면 [서비스 이용 현황] 탭 메뉴를 클릭해 주십시오.
6. 리전에서 확인할 리전을 클릭해 주십시오.
7. 필요한 정보를 확인해 주십시오.
8. 서비스: 선택한 리전에서 사용 중인 서비스의 이름
9. Region: 서비스 이용 현황을 확인하기 위해 선택한 리전 정보
10. 보유 계약 수: 선택한 리전에서 사용 중인 서비스에 대한 계약 건수
11. 콘솔 바로가기: 선택한 리전에서 사용 중인 서비스를 관리할 수 있는 콘솔로 이동하려면 클릭
